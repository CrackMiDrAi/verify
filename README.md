## 开发

请先安装好 NodeJS 环境，然后：

```shell
git clone https://github.com/CrackMiDrAi/verify.git
cd verify

npm install # 你也可以使用自己喜欢的包管理器
npm run dev
npm run lint # 写完代码保存后、提交 Commit 前，请首先运行一遍以检查代码格式是否有问题
```
